Addenda Liverpool for invoices

This module add the addenda Liverpool in invoices and credits notes. We have to set up some fields in company, partner and invoice for correct functionality

In the company set up a field.

Number GLN: Specifies the number of location global for the company.

For the partner the are 3 fields to set up:

Number GLN: In this case specifies for the buyer.

Buyer contact: Specifies information about the buyer contact.

ID supplier: Specifies the code to identify what type secundary identification is assign to supplier.

In the invoice set up 3 fields.

Purchase order: The purchase order from the partner.

Delivery folio: Specifies the number publication. This numer is emmited by the buyer when received the products.

Date delivery: Specifies the date assigned number.
